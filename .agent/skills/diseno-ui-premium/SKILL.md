# Diseño UI Premium

## Cuándo usar esta habilidad
- Cuando debas maquetar, crear CSS, o estructurar la interfaz de usuario de cualquier componente interactivo web o página completa.
- Cuando la tarea o usuario busque generar un "efecto WoW" o pida que la web se vea "moderna", "premium" o "altamente atractiva".

## Instrucciones paso a paso
1. **Definición de Colores**: Evitar colores genéricos (ej. azul plano, rojo puro). Definir en su lugar una paleta de colores armoniosa, preferiblemente basándose en el modelo de color HSL y contemplar en lo posible paletas de temas oscuros (Dark Mode).
2. **Tipografía Moderna**: Implementar web fonts modernas (ej. "Inter", "Roboto", "Outfit" u otras disponibles vía Google Fonts). Nunca depender de la tipografía por defecto del navegador web.
3. **Efectos de Vanguardia**: Usar gradientes fluidos y "glassmorphism" (desenfoques "blur", transparencias dinámicas) donde agreguen valor sin saturar visualmente al usuario.
4. **Interactividad Dinámica**: Añadir a cada botón o tarjeta interactiva efectos "hover", y pequeñas transiciones preestablecidas o micro-animaciones en acciones relevantes. Las interfaces deben sentirse "vivas" al ser recorridas con el puntero.

## Reglas y Convenciones
- NUNCA entregar el diseño de un "Minimum Viable Product" si cuentas con libertad creativa, tu objetivo principal es impresionar a nivel estético.
- NO depender de imágenes "placeholders" feas o aburridas, genera demostraciones o layouts visualmente efectivos.
